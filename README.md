Seamless Care presents an infographic about seamless care solutions. 
The project is an infographic version of the Guidebook - Seamless Care: Integrating Mental Health Services into Primary Heealthcare in the Philippines.' 

### Main Function Points
- Display seamless care infographic and visual information
- Present a three-layer solution architecture for care services
- Provide static web pages accessible via GitHub Pages deployment
- Serve as an informational resource about integrated care approaches

### Technology Stack
- HTML (100% of codebase)
- GitHub Pages (for hosting and deployment)
- Static website architecture
